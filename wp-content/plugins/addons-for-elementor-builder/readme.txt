== External services ==

= Google Maps Platform =
Vertex Addons can load the Google Maps JavaScript and Static Maps APIs hosted at `maps.googleapis.com` when you enable map-related widgets. The visitor's browser calls Google's servers with the API key you configure, the map coordinates you supply, and standard browser metadata (such as IP address and user agent) so the map tiles and related functionality can be rendered. See the [Google Maps Platform Terms of Service](https://maps.google.com/help/terms_maps/) and the [Google Privacy Policy](https://policies.google.com/privacy) for details on how Google handles this data.

= Vertex Addons CDN and API =
When you browse or import Templates Kits, the plugin connects to Webilia-operated endpoints at `cdn.webilia.com` and `api.webilia.com` to fetch kit listings, previews, and assets. Those requests transmit the selected kit identifier together with the default WordPress request headers, which include your site's URL and WordPress version, so the templates can be downloaded and installed. Review the [Vertex Addons Terms & Conditions](https://vertexaddons.com/terms-and-conditions/) and [Vertex Addons Privacy Policy](https://vertexaddons.com/privacy-policy/) to understand how that data is processed.

== Installation ==
Follow these steps to install **Vertex Addons for Elementor** on your WordPress website:

**✅ Method 1: Install via WordPress Dashboard**

1. Log in to your **WordPress admin dashboard**.
2. Navigate to **Plugins > Add New**.
3. In the search field, type **"Vertex Addons for Elementor"**.
4. Click the **Install Now** button next to the plugin.
5. Once the installation is complete, click **Activate**.

**✅ Method 2: Upload the Plugin Manually**

1. Download the **Vertex Addons for Elementor** `.zip` file from this page.
2. Log in to your **WordPress admin dashboard**.
3. Go to **Plugins > Add New > Upload Plugin**.
4. Click **Choose File**, select the downloaded `.zip` file, and click **Install Now**.
5. After installation, click **Activate Plugin**.

**What’s Next?**

- After activation, a new **"Vertex Addons"** menu will appear in your dashboard.
- Start using our powerful **widgets** and **extensions** with Elementor.

== Frequently Asked Questions ==

= Is Vertex Addons compatible with Elementor Pro? =
Yes, Vertex Addons works perfectly with both **Elementor Free** and **Elementor Pro**. You can take full advantage of its widgets and extensions regardless of which version you're using.

= How can I get support? =
You can contact our support team through the [Vertex Support](https://vertexaddons.com/support/) page.

= Can I use these widgets with other page builders? =
No, Vertex Addons is currently built specifically for **Elementor** and is not compatible with other page builders like WPBakery or Beaver Builder. Support for other builders may be considered in the future.

= Will Vertex Addons slow down my website? =
Not at all. Vertex Addons is built with performance in mind. Only the widgets you use are loaded, ensuring your site stays fast and lightweight.

= Do I need coding skills to use Vertex Addons? =
No coding is required! All widgets and features are fully customizable through the **Elementor interface** with simple drag-and-drop controls.

= Can I use Vertex Addons on multiple websites? =
Yes, the free version can be used on unlimited websites. If you're using a premium version, please refer to the license terms on our website.
