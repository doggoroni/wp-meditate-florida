Listdomer Core plugin adds some awesome functionality to Webilia Listdomer Theme.

== Description ==

If you're using Webilia Listdomer theme, then you can install this plugin to activate demo import and many features for your theme.

== External services ==

This plugin downloads optional demo content packages from Webilia when you use the Demo Importer. When you trigger an import, WordPress requests the necessary files (content XML, widgets, Elementor templates, and related assets) from the Webilia CDN at https://cdn.webilia.com. These requests include only the technical details needed to serve the files (such as your server's IP address and standard HTTP headers); no additional data from your site is transmitted. You can review Webilia's terms of service at https://webilia.com/terms-and-conditions/ and privacy policy at https://webilia.com/privacy-policy/.

== Installation ==

For installing Listdomer Core please follow below steps:

1. Install it either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
